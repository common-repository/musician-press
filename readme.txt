== Description ==

Musician Press is video entertainment for musicians. 24/7 programming. Includes daily content for bass players, clarinet players, drummers, flute, guitar, keyboard, saxophone, trumpet, violin, and singers. Musician Press is powered by Musician TV.

Features: Auto sizing so content fits perfectly inline with existing sidebar content. Works with smart / mobile devices. 

Disclosure: Musician Press relies on video content provided by and hosted by YouTube (https://youtube.com), at all times. Before using this plugin please be sure you have read and understand YouTube's terms of service and privacy policies, found here - https://www.youtube.com/static?template=terms and here - https://www.youtube.com/howyoutubeworks/our-commitments/protecting-user-data/

== Installation ==
1. Download and unzip plugin file
2. Upload the FOLDER containing the zip file contents to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. To add the Musician Press widget on a sidebar, browse to `Appearance > Widgets` and add 'Musician Press' to desired sidebar. Configure parameters like 'Title' and save your changes.
5. To add the Musician Press widget to any post or page you will probably need another plugin such as the 'Widgets in pages' plugin. See 'Additional Info' below.

== Frequently Asked Questions ==

= Can I customize / skin the Musician Press widget? =

There is currently no direct support for dynamic skinning or color schemes. All style parameters and supported features are controlled exclusively by Musician Press.
